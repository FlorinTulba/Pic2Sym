The application will generate a text representation of a provided image.

Desired Features
----------------
- choice of font type and size and character set
- selecting the size of output (width and height expressed in characters)
- optimal background & foreground for each character
- the output presented as html / image / view


Limitations
-----------
- Only fixed-width fonts are used for the transformation
- Console output was not considered, as the background and foreground will be limited to only a couple of colors


Implemented Features
--------------------
None yet.
